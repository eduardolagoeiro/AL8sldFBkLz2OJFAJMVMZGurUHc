## Instalação

```bash
yarn install
```

## Como rodar

```bash
yarn start
```

Porta 8082. O Mock (MSW) é habilitado automaticamente em dev.

```bash
yarn test        # testes (watch)
yarn test:ci     # testes (CI)
```
